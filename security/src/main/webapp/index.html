the following is an example for tomcat in windows:
  • have your JDK7 in your path
  • check file doesn't exist: C:\Users\[userName]\.keystore
  • run this command:
  • keytool -genkeypair -keyalg RSA -alias selfsigned -validity 1080
  • change tomcat setting on $CATALINA_BASE/conf/server.xml:
  • search 'Define a SSL HTTP/1.1 Connector', and follow the description to set ssl to your newly created one
