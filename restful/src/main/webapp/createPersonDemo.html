This is a demo web app to show you how to validate data

  • Web UI send data to the back end on Restful API
  • The validation happens to the back end
  • If validation fails, back end return 400 error with fields related description
  • And Web UI will apply those errors desc to UI fields

Design notes

  • Validator is on JSR 303
  • UI is on EmberJS
  • Restful API is on JSR330 and Spring
  • To make it simple, I didn't implement i18n
